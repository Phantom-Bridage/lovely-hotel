## DB Design
### 予約
* 予約番号(PK): 整数
* 予約した顧客名(FK: 顧客番号): 整数(追加)
* プラン番号(FK: プラン番号): 整数
* チェックインの日付(１５時から２３時まで): 文字列
* チェックインの時間: 整数(変更前) -> 文字列(変更後)
* チェックアウトの日付: 文字列

### 顧客
* 顧客番号(PK): 整数
* 顧客の名前: 文字列
* 顧客のメールアドレス: 文字列
* 顧客の住所: 文字列

### プラン
* プラン番号(PK): 整数
* プランの名前: 文字列
* プランの値段: 実数？
* 部屋の収容可能人数: 整数
* ベッドの種類: 文字列
* プランの特典(e.g. ミネラルウォータ付き): 文字列
